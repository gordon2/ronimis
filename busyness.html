How this is built

A collector polls each gym's live headcount every 2 minutes and logs it. This page takes every reading in the selected period, groups it by weekday and hour, and shows the average for each slot — so one cell is the typical number of people at, say, Tuesday 19:00. It is a single blended "typical week", not one specific week.

  • Times are shown in Europe/Tallinn.
  • Colour is scaled to each gym's own busiest slot, so every gym's pattern is readable; the number in the cell is the actual average headcount.
  • A faint “·” cell means no readings were collected for that slot in the selected period.
  • All data blends everything together (e.g. busy autumn with quiet winter); pick a single month to see just that month's pattern.
